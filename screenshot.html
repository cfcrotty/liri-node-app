User can run the JavaScript using `node liri.js`

Select one command(concert-this, spotify-this-song, movie-this, do-what-it-says) using the arrow keys.

After selecting a command, enter the artist/band name, song, or movie or press enter to continue.

It will show the results on the terminal.

It will also open an HTML file on your default browser.

If there is no result, it will display "Sorry! No result found.".
